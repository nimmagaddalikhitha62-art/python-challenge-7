Personal Logic

For this program, I followed a slightly different logic order based on my thinking.
Since my register number ends with 8, I decided to check total energy consumption first because overall usage gives a quick idea about efficiency.

After that, I checked for high consumption readings to detect overuse.
Then I compared efficient and moderate values to see if the usage is balanced.

I used this order because I felt checking the overall impact first is more meaningful than checking individual categories.
This made my solution a bit different and more practical in real-life scenarios.
